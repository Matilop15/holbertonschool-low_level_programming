# holbertonschool-low_level_programming
## 0x1A-hash_tables

## Purpose
- What is a hash function
- What makes a good hash function
- What is a hash table, how do they work and how to use them
- What is a collision and what are the main ways of dealing with collisions in the context of a hash table
- What are the advantages and drawbacks of using hash tables
- What are the most common use cases of hash table

## Installation
To use, first download  this repository into your local machine by issuing the following command in your local terminal. 
```
git clone https://github.com/Matilop15/holbertonschool-higher_level_programming/tree/master/0x09-python-everything_is_object
```

#### Change Directory
```
cd holbertonschool-low_level_programming/0x1A-hash_tables/
```

## Usage
```
gcc <filename.c>
```
or
```
./<filename.c>
```
